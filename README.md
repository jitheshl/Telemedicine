# AuraTelemed - AI-Powered Telemedicine Portal

A production-ready full-stack Telemedicine Consultation Portal built with the MERN stack (MongoDB, Express, React, Node.js), Tailwind CSS v4, and Google Gemini LLM API integration.

## Key Features
- **Secure Authentication**: JWT-based auth with encrypted password storage (bcryptjs) and route guards for Patient, Doctor, and Admin roles.
- **AI Symptom Checker**: Triage assistant powered by Google Gemini that analyzes patient symptoms, assesses urgency severity (Low/Medium/High), and matches patients to clinical departments.
- **AI Recommendation Engine**: Promotes matching specialized doctors based on symptom checkers.
- **Aura AI Healthcare Chatbot**: Empathetic health assistant supporting 24/7 client navigation, basic FAQ checks, and wellness tips.
- **AI Consultation Summarizer**: Clinician session recap generator that drafts treatment actions and lifestyle recovery steps automatically when doctors log patient notes.
- **Dashboards**:
  - **Patient**: Booking logs, medical histories, star review submissions, AI summaries.
  - **Doctor**: Appointment accept/cancel workflow, complete session actions, professional analytics (total sessions, earnings, rating).
  - **Admin**: System-wide dashboard, specialty bar charts, doctor verification.

## Getting Started

### Prerequisites
- Node.js installed (v18+ recommended)
- MongoDB Atlas connection string (or running local MongoDB)
- Google Gemini API key (optional; has local fallback engine)

### Setup backend:
1. Navigate to the `backend/` folder:
   ```bash
   cd backend
   ```
2. Create a `.env` file from the template:
   ```bash
   cp .env.example .env
   ```
3. Set your environment variables in `.env`:
   - `MONGODB_URI`: your MongoDB Atlas connection string
   - `GEMINI_API_KEY`: your Google Generative AI API key
   - `JWT_SECRET`: any secure secret string
4. Install dependencies:
   ```bash
   npm install
   ```
5. Start backend development server:
   ```bash
   npm run dev
   ```
   *(Running on port `5000`)*

### Setup frontend:
1. Navigate to the `frontend/` folder:
   ```bash
   cd ../frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start Vite client server:
   ```bash
   npm run dev
   ```
   *(Running on port `5173` with proxy set to port `5000`)*

## Testing Workflow
1. **Register User**: Register a Patient account on `/register`.
2. **AI Symptom Check**: Open the Symptom Checker from the patient dashboard or Home page. Input `I have constant chest pain and heart racing.` to test the high-severity Cardiologist router, or `I have minor skin itching` for dermatologist.
3. **Book Doctor**: Select a doctor slot, input symptoms, and schedule booking.
4. **Accept & Complete**:
   - Register a Doctor account.
   - Access the Admin Dashboard (requires Admin role set in DB) to verify the doctor.
   - Log in as the doctor, accept the booking, and click **Complete Session** (add diagnostic notes).
5. **View AI Summary**: Log back in as the patient and view the detailed consultation summary generated by Gemini.
